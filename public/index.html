Jockpack

A webpack 3 build system to simplify the use of various javascript technologies !

Features

Stable

  • ES6 - ES7 transpiles to ES5
  • Code splitting, dead code removal & code minification
  • Automated assets manifest publication
  • Use SCSS
  • Use Vue.js
  • Assets preloading
  • WebFonts async loading
  • Images lazy loading
  • CSS async loading
  • Dev tools
  • CLI Image minifier
  • IE11 compatibility compilation tool
  • Typescript support
  • Pre-existing Linter configuration
  • Webpack 3 build config wrapper for ease of use
Experimental
  • Progressive web App assets & manifest generation
  • Easy critical css configuration

If that's the case for any of it, Welcome !
